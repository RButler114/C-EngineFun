Detailed Description

Comprehensive input handling system for keyboard and mouse input.

The InputManager provides a high-level interface for handling user input, supporting both current state queries and edge detection (just pressed/released). It maintains both current and previous frame state to enable proper event detection.

Features:

  • Keyboard state tracking with edge detection
  • Mouse button state tracking with edge detection
  • Mouse position and movement tracking
  • Mouse wheel support
  • Cursor visibility control
  • Frame-based state management
