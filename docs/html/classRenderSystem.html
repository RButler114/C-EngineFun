Detailed Description

System that renders entities with TransformComponent and RenderComponent.

The RenderSystem processes all entities that have both TransformComponent and RenderComponent, drawing them on screen using the provided Renderer.

Features:

  • Renders colored rectangles based on component data
  • Respects visibility flags
  • Uses transform position for rendering location
  • Supports RGBA color values
