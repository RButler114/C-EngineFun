Detailed Description

Abstract base class for all ECS systems.

Systems contain the logic that operates on entities with specific components. Each system maintains a set of entities it's interested in and processes them during the Update() call.

Systems are notified when entities are added or removed from the world, allowing them to maintain their own entity lists based on component requirements.
